# 감리일정 분석기.

포탈에서 저장한 HTML 파일을 넣으면 이용 가능.
포탈 들어가서 Ctrl+S 하시면 HTML 저장 가능합니다.

### 1. 시정조치결과확인
- 기본적으로 시정조치결과가 0을 초과하는 경우 강조표시가 되어 나오긴 하지만 좌측 하단의 [시정조치결과 확인] 버튼을 눌러서도 강조 가능합니다.

### 2. 전문가 직접 입력
- 감리원 및 전문가 투입 비율 계산용. 감리원이 아닌 전문가분들을 적으면 추가 전문가-나머지 제안란에 전문가 MD를 뺀 공수가 나옵니다.
- ENTER: 적용, SHIFT ENTER: 줄바꿈

### 3. 인력 모음- 감리원 등급
- 포탈의 감리원 등급에 영향을 받아 표기됩니다.
- 테스터는 전문가로 표기됩니다.
- 전문가임에도 감리원증에 번호가 존재하는 경우 확인필요로 뜹니다.

### 4. 비교 기능
- JSON 저장 버튼을 누르면 현재 띄워놓은 페이지가 다운로드 폴더에 저장되며, 캐시 또한 페이지에 저장됩니다.
- 페이지를 닫은 후 다시 페이지를 띄우실 때 컴퓨터에 저장된 JSON 파일을 불러오거나, 로컬 캐시를 클릭하여 열 수 있습니다.
- <img width="706" height="430" alt="image" src="https://github.com/user-attachments/assets/a3bc4459-3694-4e17-8f2b-31f87cce88bb" />
- 페이지를 띄운 상태에서 포탈에서 새로 저장한 HTML 파일을 비교 파일란에 업로드하면 해당 파일과 비교해서 달라진 내용을 강조해줍니다.
- 해당 내용은 마우스 커서를 올리면 대략적인 파악이 가능합니다.
- 현재까지 [인력의 공수 변경], [인력 추가 및 삭제], [총 공수 변경], [일정 변경] 내용 강조 확인되었습니다.
